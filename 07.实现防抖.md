# 08. 手撕防抖

```html
<button>按钮</button>
```

初级版本：
第一个参数是事件触发的函数，第二个参数是防抖延迟时间
```javascript
function debounce(fn, delay) {
    let timer;
    return function() {
        let args = arguments;
        clearTimeout(timer);
        timer = setTimeout(() => {
            fn.apply(this, args);
        }, delay);
    }
}
let input = document.querySelector('input');
input.oninput = debounce(function() {
    console.log(this, input.value);
}, 1000)
```
